# Calories-Info-Bot
By typing the name of the desired food product, you can use this robot to give you the amount of :

1-calories

2-protein

3-total fat

4-saturated fat

5-total carbohydrate

6-fiber

7-sugar

8-sodium

9-potassium

10-cholesterol

contained in that food product.

You can also get information about each of the above.
